# Code of Conduct

## Our Standards

Participants are expected to be respectful, constructive, and professional.

Unacceptable behavior includes:
- Harassment or discrimination
- Personal attacks
- Trolling or bad-faith disruption

## Scope

This Code of Conduct applies to project spaces and public interactions representing this project.

## Enforcement

Report concerns to `mblackshear@gmail.com`.
Maintainers may remove or restrict participation for violations.
